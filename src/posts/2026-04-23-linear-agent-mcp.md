Linear 在 4/23 的 changelog 宣布 Linear Agent 正式支援 MCP（Model Context Protocol）。

簡單說：Agent 之前只能讀寫 Linear workspace 內部的資訊（issues、projects、docs），現在可以連到你工作流裡的其他工具——拉外部資料、執行外部動作、把分散的脈絡帶回 Linear 的任務和文件。

對 PM 與設計師的意義：規劃專案、查 issue、寫 update 或起草 spec 時，Agent 可以同時參考程式碼 repo、外部文件、設計稿，不只是 Linear 內部的 issue 描述。原本要在五個分頁之間切換手動拼湊脈絡的事，理論上可以一次到位。

實務效果取決於 MCP server 生態系的成熟度。Linear 把門打開，剩下的就看你接什麼工具進來。
